# 摊伴 TANBAN

面向咖啡摊、夜市摊、餐车和小型门店的一体化餐饮 SaaS 原型。

当前原型在同一入口中提供三种体验：

- 平台管理端：商户、套餐计费、小程序版本、能力中心、支付分账、工单与系统设置。
- 商户运营端：订单、商品库存、用户会员、营销、财务、装修、数据、打印与提醒。
- 顾客点单端：扫码进店、选品、购物车、微信支付模拟、取餐码与订单进度。

详细产品结构、现有系统盘点与后续工程拆分见 [docs/SYSTEM_BLUEPRINT.md](docs/SYSTEM_BLUEPRINT.md)。

## 本地运行

```bash
npm install
npm run dev
```

生产构建：

```bash
npm run build
```
